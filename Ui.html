add: function(Ui,{any : any } ):(Response)

this function just shows the text. only argument is string

This is a generic function that takes and adds a specific widget to the Ui. This takes a table as argument. below, you can see how the table will be used. The table represents a generic widget and what the fields mean will be decided by the widget itself. The table must have a field called "widget_type" representing the type of widget with any of the following values: button, custom custom is a widget which is created inside lua itself to help addon makers reuse widgets. all widgets will basically use this table and Ui to draw themselves. different widgets need different data.

Button: text: string. the text to show inside the button. wrap: bool. whether the button should wrap the inside text.

set_enabled: function(Ui,boolean):()

makes the Ui unable to be interacted with input. once set, it cannot be unset.

scope: function(Ui,function(...any):any...):((Response),(MultiValue))

new scope to make some localized changes without affect the rest of the Ui after this

add_enabled_ui: function(Ui,(boolean),(function(...any):any...)):((Response),(MultiValue))

will create a new scope and add the ui after setting whether it should be enabled or not. won't affect other Ui after this
